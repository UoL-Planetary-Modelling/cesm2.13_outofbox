# Instructions for running versions of CESM on Leeds ARC4

## ** Main Branch **
- **README:** this file

- **FX2000_SMin:** WACCM-X for perpetual year 2000 - Solar Min conditions (f10.7=70, f10.7a=70, kp=0.3, ap=1.8)
